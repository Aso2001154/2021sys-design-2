   }

    [*] --> カートの中ページ

    カートの中ページ -> 注文確定ページ:「注文確定」をクリック 
    注文確定ページ --> [*]: 「商品一覧へ戻る」をクリック
   }
   カートの中 -up[dotted]-> 商品一覧:「詳細へ」クリック
   商品一覧 -right-> 登録情報
   
 }
