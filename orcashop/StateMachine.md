```uml
@startuml

[*] -> メインページ: 最初の画面
メインページ -> 商品一覧ページ: 商品検索、ジャンル選択
メインページ -> 商品詳細ページ: おすすめ商品など商品画像クリック
商品一覧ページ -> 商品詳細ページ: 商品画像クリック
商品詳細ページ -down-> カートページ: カートを見るをクリック
カートページ -> 注文ページ: 注文に進むをクリック
メインページ -down-> アカウント情報ページ: アカウント情報をクリック
メインページ -> カートページ: カートを見るをクリック
アカウント情報ページ -down-> アカウント情報登録編集ページ: 登録内容を編集するをクリック
アカウント情報登録編集ページ -left-> アカウント情報ページ: 戻る
メインページ -up-> ログインページ: ログインをクリック
ログインページ -> メインページ: 戻る

@enduml
```
